**Rakuen** est une plateforme web immersive conçue pour les passionnés d'anime, offrant un suivi précis, des calendriers de sortie en temps réel et une recherche intelligente ultra-performante.

## 🚀 Le projet
Le projet subit actuellement une refonte majeure pour la version **3.4.2**. 
L'objectif est de proposer une expérience utilisateur digne des plus grandes plateformes comme Crunchyroll, avec une attention particulière portée à la fluidité et à la richesse visuelle.

### ✨ Fonctionnalités clés
- **Interface Immersive** : Design "Cinema Mode" avec bannières dynamiques et sections horizontales.
- **Smart Search v3** : Algorithme de recherche floue (Fuzzy Search) via Fuse.js.
- **Saga Tracker** : Suivi complet de sagas complexes comme *Tensei Shitara Slime Datta Ken*.
- **Offline Ready** : Gestion intelligente du cache et des favoris locaux.

## 🚧 État actuel
> **"Le gros arrive après."**

Le code source est actuellement maintenu dans un dépôt privé durant cette phase de transition. La page d'accueil actuelle sert de vitrine technologique et de point d'attente pour la communauté.

## 🛠️ Technologies
- **Frontend** : HTML5, Tailwind CSS, JavaScript (ES6+)
- **Logic** : Fuse.js (Intelligence Artificielle de recherche)
- **Data** : Jikan API (MyAnimeList)
